Sum of Two

Language: Java

A simple way to get sum of two numbers.


public class SumOfTwoNumbers {
    public static void main(String[] args) {
        // Declare and initialize two integer variables
        int num1 = 15, num2 = 25;

        // Calculate the sum
        int sum = num1 + num2;

        // Print the result
        System.out.println("The sum of "+ num1 +" and "+ num2 +" is: "+ sum);
    }
}
